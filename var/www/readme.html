But this page explains how to access git repositories hosted on this server.

Usage

Pulling

If you are not Tom, this is probably all you'll be doing.

Creating

cgit searches for repositories in /home/tlevine. Here's a little helper to create repositories.

git-create() {
  repo=$(basename "$1" .git)
  ssh tlevine@git.thomaslevine.com "git-create-server" &&
    echo "Success! Now try this:" &&
    echo "git remote add origin 'tlevine@git.thomaslevine.com:$repo.git'"
}

Run it like this

git-create foo

You may also edit the cgitrc file inside of your repository if you want things to look nice.

Pushing

Set up SSH keys if you want.

scp-copy-id tlevine@git.thomaslevine.com

Then set the remote.

git remote add cgit tlevine@git.thomaslevine.com:$(basename "$PWD") master
