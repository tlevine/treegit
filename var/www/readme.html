About Tom's git hosting

Tom hosts his git repositories on this server. He also hosts them on some other places

  • GitHub

But this page explains how to access git repositories hosted on this server.

Usage

Pulling

If you are not Tom, this is probably all you'll be doing.

Creating

cgit searches for repositories in /home/TREEGIT_USERNAME. Here's a little helper to create repositories.

treegit-create() {
  ssh TREEGIT_USERNAME@TREEGIT_DOMAIN_NAME "treegit-create '$1'"
}

Run it like this

treegit-create $(basename "$PWD")

You may also edit the cgitrc file inside of your repository if you want things to look nice.

Pushing

Set up SSH keys if you want.

scp-copy-id TREEGIT_USERNAME@TREEGIT_DOMAIN_NAME

Then set the remote.

git remote add cgit TREEGIT_USERNAME@TREEGIT_DOMAIN_NAME:$(basename "$PWD").git master
