About Tom's git hosting

Tom hosts his git repositories on this server. He also hosts them on some other places

  • GitHub
  • Gitorious

But this page explains how to access git repositories hosted on this server.

Usage

Pulling

There is a "Clone" url at the bottom of each repository's summary page. git clone that.

Creating

cgit searches for repositories in /home/TREEGIT_USERNAME. Here's a little helper to create repositories.

treegit-create() {
  ssh TREEGIT_USERNAME@TREEGIT_DOMAIN_NAME "treegit-create '$1'"
}

Run it like this

treegit-create $(basename "$PWD")

Organizing repositories

If you add a readme.html or a readme.md file to the root of the repository, it will show up in the repository's "about" tab.

Once you've made a readme, edit the cgitrc file inside of your repository if you want things to look even nicer. Consider setting these at least:

desc=An off-the-self cgit server with some helpers.
section=boxes

This sets a description to appear next to the repository title and a section in which to list the repository.

Pushing

Set up SSH keys if you want.

scp-copy-id TREEGIT_USERNAME@TREEGIT_DOMAIN_NAME

Then set the remote.

git remote add cgit TREEGIT_USERNAME@TREEGIT_DOMAIN_NAME:$(basename "$PWD").git master
